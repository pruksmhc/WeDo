
How to run:
-Start the rethinkdb server by calling "rethinkdb", then open another command line tab and then run node server.js
https://www.rethinkdb.com/docs/start-a-server/
-Go to http://www.wedoappbeta.com , and if you have access, you'll be able to log in and start using it.
